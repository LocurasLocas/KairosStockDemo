Cómo conectar una plataforma
🛵 Rappi
  1. Ingresá al Portal de Aliados Rappi
  2. Ve a Integraciones → API
  3. Generá tus credenciales
  4. Copiá el Store ID y el Client Secret
  5. Pegá los datos arriba y hacé clic en Conectar
  6. Configurá la URL del webhook en Rappi
🚲 PedidosYa
  1. Accedé al Panel de Socios PedidosYa
  2. Ve a Configuración → API / Integraciones
  3. Creá una nueva integración
  4. Copiá el Client ID, Secret y Restaurant ID
  5. Pegá los datos arriba y hacé clic en Conectar
  6. Configurá la URL del webhook en PedidosYa
